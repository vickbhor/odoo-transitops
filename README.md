# TransitOps — Fleet Operations Platform

Hackathon build. Stack: Next.js (App Router, JavaScript) + Tailwind CSS + Supabase (Postgres + Auth).

## Setup
1. `npm install`
2. Copy `.env.example` to `.env.local` and fill in your Supabase project URL + anon key
3. `npm run dev`

## Track ownership
- Leader: Vehicles, Maintenance, Fuel & Expenses, Reports
- Teammate: Auth/RBAC, Layout/Dashboard, Drivers, Trip lifecycle
